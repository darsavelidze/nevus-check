# Nevus Analyzer - Анализатор кожных новообразований

Веб-приложение для классификации кожных новообразований с использованием глубокого обучения на базе датасета HAM10000.

## 🚀 Быстрый старт

### 1. Установка зависимостей

```powershell
pip install -r requirements.txt
```

### 2. Обучение модели (опционально)

Если нужно обучить модель заново:

```powershell
python training/train.py
```

### 3. Запуск веб-приложения

```powershell
python run.py
```

Приложение будет доступно по адресу: http://localhost:5000

## 📊 Классы кожных новообразований

- **akiec**: Актинический кератоз / Болезнь Боуэна
- **bcc**: Базально-клеточная карцинома
- **bkl**: Доброкачественный кератоз
- **df**: Дерматофиброма
- **mel**: Меланома
- **nv**: Меланоцитарный невус
- **vasc**: Сосудистое поражение

## 🔧 Основные компоненты

### Flask Application (`app/`)

- **`__init__.py`**: Создает и конфигурирует Flask приложение
- **`routes.py`**: Определяет эндпоинты API:
  - `GET /` - главная страница
  - `POST /predict` - предсказание на загруженном изображении
  - `GET /health` - проверка работоспособности
- **`model_loader.py`**: Класс `SkinLesionModel` для загрузки и использования модели
- **`utils.py`**: Функции предобработки изображений и валидации

### Training (`training/`)

- **`train.py`**: Полный цикл обучения модели с гиперпараметрической оптимизацией
- **`model_manager.py`**: Управление версиями моделей и их метаданными

### Models (`models/`)

Директория для хранения:
- Весов обученных моделей (`.h5` файлы)
- Гиперпараметров (`best_hparams.json`)
- Графиков обучения (в `plots/`)

## 📝 API Endpoints

### POST /predict

Загружает изображение и возвращает предсказание.

**Request:**
- Method: POST
- Content-Type: multipart/form-data
- Body: file (изображение)

**Response:**
```json
{
  "success": true,
  "predicted_class": "nv",
  "predicted_description": "Меланоцитарный невус",
  "confidence": 0.95,
  "confidence_percentage": "95.00%",
  "all_predictions": [
    {
      "class": "nv",
      "description": "Меланоцитарный невус",
      "probability": 0.95,
      "percentage": "95.00%"
    },
    ...
  ],
  "image_path": "/static/uploads/image.jpg"
}
```

### GET /health

Проверка состояния приложения.

**Response:**
```json
{
  "status": "ok",
  "model_loaded": true
}
```

## ⚠️ Важные замечания

1. **Медицинское использование**: Это приложение создано только для образовательных целей. Результаты не являются медицинским диагнозом.

2. **Данные**: Датасет HAM10000 не включен в репозиторий. Используйте `data/download_dataset.py` для загрузки.

3. **Веса модели**: Файлы весов (.h5) исключены из git. После обучения они сохраняются в `models/`.

4. **Устаревший код**: Файл `app.py` в корне оставлен для обратной совместимости. Используйте `run.py`.

## 🔨 Разработка

### Добавление новых маршрутов

Добавьте маршруты в `app/routes.py`:

```python
@main_bp.route('/new-route')
def new_route():
    return jsonify({'message': 'Hello'})
```

### Модификация модели

1. Измените архитектуру в `app/model_loader.py` и `training/train.py`
2. Переобучите модель: `python training/train.py`
3. Перезапустите веб-приложение
